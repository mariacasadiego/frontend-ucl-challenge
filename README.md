# 🏆 Champions League Draw – Frontend

Frontend application built with React and TypeScript to visualize and explore Champions League fixtures generated by the backend API.

## Features

### Fixtures

* General fixtures listing
* Pagination
* Responsive layout

### Team View

* View fixtures for a selected team
* Home and away fixtures
* Team statistics summary

### Matchday View

* View fixtures grouped by matchday
* Quick navigation between matchdays

### Filters

* Team filter
* Matchday filter
* Country filter
* Venue filter (Home / Away)
* Active filter tags
* Clear filters action

### User Experience

* Loading state
* Error state
* Empty state
* Responsive design
* Accessible controls and navigation

## Architecture

The project follows a feature-based structure:

```txt
src/
├── app/
├── api/
├── features/
│   └── matches/
├── shared/
│   ├── components/
│   └── styles/
└── router/
```

This approach keeps business features isolated and improves maintainability and scalability.

---

## Running the Project

Install dependencies:

```bash
npm install
```

Configure environment variables:

```env
VITE_API_URL=http://localhost:8000
```

Start development server:

```bash
npm run dev
```

Create production build:

```bash
npm run build
```

---

## Technical Decisions

### State Management

TanStack Query is used for server state management, caching and asynchronous API requests.

### Styling

SCSS Modules were selected to keep styles isolated and maintainable.

### Filtering

Filtering is performed on the client side after fetching fixture data, providing a fast and responsive user experience.

### Accessibility

The interface includes semantic elements, keyboard-friendly interactions and descriptive labels to improve accessibility.
